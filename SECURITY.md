# Security Policy

## 🛡️ Supported Versions

The following versions of this project are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅ Yes             |
| < 1.0   | ❌ No              |

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

### 📩 How to Report
- Email: krabhishek0321@gmail.com  
- Or open a GitHub issue (if the vulnerability is not sensitive)

### ⚠️ Guidelines
- Do not publicly disclose the vulnerability before reporting
- Provide detailed steps to reproduce the issue
- Include screenshots or code snippets if possible

---

## ⏱️ Response Time
- Initial response within **48-72 hours**
- Fixes or updates will be provided as soon as possible

---

## ✅ What to Expect
- Acknowledgement of the issue
- Investigation and validation
- Fix implementation (if valid)
- Notification once resolved

---

## ❌ Out of Scope
- Issues related to third-party services (e.g., hosting providers)
- General bugs that are not security-related

---

💡 Thank you for helping improve the security of this project!
